## How Hermes Was Used

This game was designed by Hermes Agent. I gave Hermes a three-word creative prompt and Hermes returned the full game concept.

### The Prompt
Make a tiny arcade game from 3 words: neon courier memory.
Return title, rules, visual style, and core loop only.

### Hermes Response
TITLE: NEON ECHO

RULES:
1. Navigate a dense futuristic city delivering glowing Memory Cores.
2. City layout is visible for 3 seconds at the start of each level, then goes dark.
3. Walls are invisible unless you use a Pulse (limited) or hit them.
4. Deliver cores to neon drop zones before the timer expires.

VISUAL STYLE:
High-contrast 8-bit aesthetic. Pitch black backgrounds with vibrant cyan, magenta, and lime-green vector lines.

CORE LOOP:
Memorize Map -> Sprint blindly through darkness -> Pulse to verify path -> Deliver Memory -> Upgrade.

## How I Adapted It

Hermes proposed a complex memory-maze concept. I distilled it into a playable arcade prototype with the same neon cyberpunk DNA.

## Controls
- Move: WASD or Arrow keys
- Start/Restart: Space
- Goal: Collect blue Memory Cores in 45 seconds
- Avoid: Red Signal Blockers
- Bonus: Gold Sync Boosts (4-second shield)

## Tech Stack
HTML5 Canvas, Vanilla JavaScript, CSS, Vercel.
