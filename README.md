## How I Adapted It

Hermes proposed a complex memory-maze concept. I distilled it into a playable arcade prototype:
- Kept the neon cyberpunk aesthetic
- Simplified the maze into an open arena (better playability)
- Preserved core mechanics: collect memory cores, dodge blockers, time pressure
- Added gold sync boosts for shield (my creative addition)

## How to Play

- Move: WASD or Arrow keys
- Start/Restart: Space
- Goal: Collect blue Memory Cores in 45 seconds
- Avoid: Red Signal Blockers (cost time + score)
- Bonus: Gold Sync Boosts give a 4-second shield
